#EN
Go to the directory with an example.

The template and content.json files are in the /templates/bakeproject/build/html/ subdirectories.

Run grunt in the "develop" directory
Make changes in the content.json file - change "cur_section"
The .html files will be compiled in the /templates/bakeproject/ directory
